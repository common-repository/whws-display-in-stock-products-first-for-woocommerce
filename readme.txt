Display in-stock products first in WooCommerce.

== Description ==
Use this handy little plugin to change the default sort order and display in stock products first in your WooCommerce store.

== Installation ==
This section describes how to install the plugin and get it working.

1. Download and install the plugin from WordPress dashboard.
2. Activate the plugin through the 'Plugins' menu in WordPress.
